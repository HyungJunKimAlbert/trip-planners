# ✈️ Trip Planners

여행 일정·예산·링크·사진을 한곳에서 관리하는 정적 웹 일정표입니다.

- **미국 서부** (라스베가스 · LA) — `usa/`
- **일본 후쿠오카** — `fukuoka/`

## 특징
- 서버 없이 동작하는 단일 HTML (정적)
- 일정 타임라인 · 예산(실시간 환율) · 링크 모음 · 지도 임베드
- 메모에 사진 붙여넣기(자동 압축)
- 데이터는 각 브라우저의 `localStorage`에 저장 / JSON 내보내기·가져오기 지원

## 호스팅
GitHub Pages로 배포됩니다. 데이터는 서버에 올라가지 않고 각자 브라우저에만 저장됩니다.
